# Go-webserver-demo
> 用Go实现的一个简单的Web服务器。

## Get started

1. clone
   ```bash
   git clone https://github.com/ScarboroughCoral/Go-webserver-demo.git
   ```
2. build
   ```bash
   cd Go-webserver-demo/
   go build
   ```
3. run
   ```bash
   ./webserver
   ```
